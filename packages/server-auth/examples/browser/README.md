# Browser Authentication Example

This example demonstrates how to authenticate with a Caido instance using browser-based authentication.

## Usage

1. Install dependencies:

   ```bash
   pnpm install
   ```

2. Run the example:

   ```bash
   pnpm run
   ```

3. Follow the instructions displayed in the console to complete authentication in your browser.

## Environment Variables

- `CAIDO_INSTANCE_URL` - The URL of your Caido instance (default: `http://localhost:8080`)
